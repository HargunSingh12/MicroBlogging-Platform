# MicroBlogging Platofrm

A social media app with user authentication, blog posting, and dynamic timeline generation using MongoDB aggregation.

## Features
- User registration and login
- Blog creation and like functionality
- Profile view and timeline feed
- Efficient querying using MongoDB aggregation

## Tech Stack
- MongoDB (+ Aggregation)
- Express.js
- React.js
- Node.js

## Notes
- Aggregation used to efficiently fetch and join blogs with user data
- Authentication handled using JWT tokens
